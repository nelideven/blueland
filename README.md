# blueland
A reactive Bluetooth frontend daemon for Hyprland users.

## Dependencies:
- dbus-next (pip install dbus-next)
- BlueZ
- Python 3.9 or higher

## Usage

1. Clone the repo:
```
bash
git clone https://github.com/yourname/blueland.git
cd blueland
```

2. Run the backend
```
python blueland.py
```
